# GoldX – High-Performance XAUUSD Trading Engine

GoldX is OrgaX's flagship trading engine designed for high-frequency scalping on **XAUUSD**.  
It leverages multi-timeframe trend validation, volatility-aware execution, and dynamic risk controls.

This repository provides:
- Public SDK structure  
- Documentation  
- API references  
- Examples (non-sensitive)  
- Integration guidelines  

> **Note:** This repo contains *public* information only.  
> The full proprietary engine is private inside OrgaX's infrastructure.

---

## Key Features
- Multi-timeframe trend detection (M1 → H1)  
- Volatility-adaptive scalping logic  
- Risk-controlled execution
